# umi

## 1.介绍

- umijs 是一个类 next.js 的 react 开发框架
- 他基于一个约定，既 pages 目录下的文件即路由，而文件则导出 react 组件
- 然后打通从源码到产物的每个阶段，并配以完善的插件体系，让我们能把 umi 的产物部署到各种场景里

![](./1.png)

## 2.新建项目

- 全局安装 umi

```sh
cnpm install -g umi
```

- 新建项目目录

```sh
mkdir umi-eg
cd umi-eg
```

- 创建 umi 项目

```sh
npm create umi
```

- 启动

```
npm start
```
