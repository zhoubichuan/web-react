# 组件开发

::: tip 前言
递归组件就是组件在模板中调用自己，需要满足两个条件：

- 在组件中设置一个`name`选项，通过这个字段拿到组件
- 加上结束条件，防止一直递归下去，抛出`max stack size exeeded`错误

:::
