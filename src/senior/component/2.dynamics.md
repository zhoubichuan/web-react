# 二.dynamics

::: tip 前言
动态组件，即组件可以作为一个变量传入到某个组件中根据一些条件，动态地切换某个组件，或动态地选择渲染某个组件
:::


::: tip 总结
合理使用动态组件可以让我们写出的代码更加简洁，减少冗余代码
:::
